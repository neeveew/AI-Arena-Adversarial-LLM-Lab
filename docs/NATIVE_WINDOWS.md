# Native Windows

The active native Windows app is the WPF app under `windows-wpf/`.

- WPF app: `windows-wpf/AI Arena - WPF.sln`
- WPF project: `windows-wpf/src/AIArena.Wpf/AIArena.Wpf.csproj`
- WPF port map: `windows-wpf/docs/WPF_PORT_MAP.md`
- Shared .NET core: `shared-dotnet/src/AIArena.Core`
- Shared .NET tests: `shared-dotnet/src/AIArena.Tests`
- Root repository map: `README.md`

The previous Python dashboard and archived WinUI shell have been removed from the active source tree. New work should target the WPF app and shared .NET core modules.
